# Introduction

Get started in c with an makefile,header and simple project structure.

# Quick start

1.Clone the repo

```bash
git clone --depth 1 https://github.com/BayonetArch/setup_c.git
```


2.Build the project

```bash
cargo build --release
```

3. run

```bash
./target/release/setup_c <project_name>
```
